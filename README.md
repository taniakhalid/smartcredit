# SmartCredit – AI Credit Risk Prediction System

SmartCredit is a machine learning-powered web application that predicts credit risk and provides interactive analytics for financial decision-making. The system helps evaluate loan applicants based on various financial and demographic factors.

## Features
- Credit risk prediction using machine learning model
- Interactive dashboard with visual analytics
- Real-time user input for predictions
- Data preprocessing and feature encoding
- Model performance evaluation
- Clean and responsive Streamlit UI

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas & NumPy
- Plotly (for visualizations)
